# 🎵 Music Visualizer – Final Project (Solved Version)

This is a custom-built interactive **Music Visualizer** made using **p5.js** and **p5.sound**, featuring five unique audio-reactive visualizations.

## 🚀 Features

- 🎚️ Toggle music playback using the button in the top-left corner
- 🔁 Switch between 7 immersive visualization modes:
  - `Spectrum`
  - `WavePattern`
  - `CircleWave`
  - `Needles`
  - `WaveRings`
  - `BarGrid`
  - `RadialBurst`
- 🔊 Real-time audio analysis using FFT and waveform data
- 🌐 Toggle fullscreen by clicking anywhere except the play button
- 🖱️ Visual effects: glow, animated waveforms, color blending, and particle trails
- 📜 Press `SPACEBAR` to open the visualization selection menu

## ✅ How to Run

1. Open the project folder in VS Code  
2. Right-click `index.html` → **Open with Live Server**  
   _(or use any local server environment)_
3. Click the **Play button** (top-left corner) to begin playback  
4. Press **SPACEBAR** to open the menu and select visualizations
